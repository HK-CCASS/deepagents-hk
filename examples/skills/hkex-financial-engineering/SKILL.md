## 背景：什么是财技？

港股中的财技是指大股东用于以下目的的公司行动和市场操纵技术：
1. **筹集资金** - 通过配售、供股
2. **巩固控制** - 通过要约、资本重组
3. **提取价值** - 通过派发模式、买卖壳游戏
4. **重置股价** - 通过合股 + 拆股组合

### 財技股5大致勝關鍵

| # | 關鍵要素 | 說明 |
|---|---------|------|
| 1 | **時機判斷** | 財技操作前後的入場/離場時機 |
| 2 | **大股東行為** | 追蹤大股東的增減持、承諾、關聯交易 |
| 3 | **條款細節** | 深入分析公告中的小字條款（折讓率、包銷、鎖定期） |
| 4 | **歷史模式** | 對比該公司/該大股東過往財技操作結果 |
| 5 | **CCASS信號** | 結合持倉變化判斷市場真實動向 |

**致勝公式:**
```
財技獲利 = 正確識別模式 × 正確時機 × 風險控制
```

### 0.1 大股东配售的真正目的

**Q1: 大股东"供股目的"是什么？**

| 目的类型 | 关键词 | 含义 | 对散户影响 |
|----------|--------|------|-----------|
| **供钱** | 錢 | 真正筹资，用于业务发展 | 中性，视项目质量 |
| **供乾** | 乾 | 赶走散户，逼你走 | 实为利好（反向操作） |
| **其他目的** | 加大市值 / 易炒 / 賣殼 / 震倉 | 财技操作准备 | 需具体分析 |

**判断公式**（Hilton核心算法）：

```
供乾信号（利好！）：
├── Project係好 → 唔想你供 → 供乾 → 向好
├── 大股东持股：(供股前持股+承诺认购)/(供股前总股+新股) > 75%
└── 散户持股 < 20%

供钱信号（利空）：
├── Project係差 → 想你供 → 供錢 → 向下
├── 大股东持股 < 30%
└── 散户承担 > 70%
```

**实战案例公式**：
```
假设：大股东持50股，供股1:2（100股变300股），大股东承诺全额认购200股

大股东占比 = (50 + 200) / (100 + 200) = 250/300 = 83.3% ✓ 供乾信号
散户占比 = 16.7%
```

### 0.2 供股5个重要日子（时间轴）

```
宣布日 ──1-2周── 除权日 ──3-5日── 最后付款日 ──3-5日── R Report ──── 生效日
  │                 │                 │                  │               │
出通告            股价自动调整       必须决定          公布认购份数      新股出埸
                 (最关键！)         是否参与           (判断供乾)
```

| # | 日期 | 英文 | 核心意义 | 策略要点 |
|---|------|------|---------|---------|
| 1 | **宣布日** | Announcement | 公布供股计划 | 判断供钱/供乾，大比例大折让→第一口价卖出 |
| 2 | **除权日** | Ex-Rights | ⭐**最重要** 股价自动调整 | 除权前买入有供股权，除权后无 |
| 3 | **最后付款日** | Last Day for Payment | 付款截止日 | 必须在此前决定是否参与，券商截止更早！ |
| 4 | **R Report** | Allotment Results | 公布认购份数 | **<30份 = 供乾信号确认** |
| 5 | **生效日** | New Shares Listing | 新股上市 | 供乾股票往往在此日开始上涨 |

### 0.3 除权日后的2种走势

#### 走势A：供钱股（利空）

```
股价
  ^
$0.3│    ╭───╮
    │   ╱     ╲    ← 最后付款日前拉高
    │  ╱       ╲   
供股价│─╱─────────╲───────> 继续下跌
$0.1│╱
    └──┼──────────┼──────────> 时间
      除权日    最后付款日
```

**特征**：
- 除权日后股价反弹远离供股价
- 最后付款日后股价崩塌
- 大股东在高位减持

#### 走势B：供乾股（利好！）

```
股价
  ^                         ╱
$0.3│                      ╱
    │                    ╱  ← 生效日后大涨
    │                  ╱
供股价│────────────────●────> 
$0.1│  ← 贴近供股价横行
    └──┼────────────┼────┼──> 时间
      除权日     最后付款日 生效日
```

**特征**：
- 除权日后股价贴近供股价横行
- 散户"认为跌到供股价，唔供就亏"，纷纷卖出
- 大股东低位收货
- R Report 显示 <30份申请 → **确认供乾**
- 生效日后股价大涨

### 0.4 供乾 vs 供钱 的交易策略

| 策略 | 买入时机 | 卖出时机 | 判断条件 |
|------|---------|---------|---------|
| **供乾** | R Report公布后/确认无人供 | 出货信号出现（半年内） | 份数<30，股价贴近供股价 |
| **供钱** | **不买！** 或除权日后短炒 | 最后付款日前1-2日 | 股价远高于供股价 |

### 0.7 财技股的3种形态

#### 1️⃣ 斗长命股（安全但无聊）

```
股价
  ^
  │  卖       父
  │  ◯───────◯───────────────────> 
  │  ← 横行不动 →
  └─────────────────────────────> 时间
```

**特征**：
1. 股价唔郁（不动）
2. 极低成交
3. 大股东持货比例好高
4. **安全**（但需要耐心等待）

#### 2️⃣ 生不如死股（陷阱！）

```
股价
  ^
  │╲
  │ ╲╲
  │  ╲╲    ← 持续下跌
  │   ╲╲
  │    ╲╲
  │     ╲╲
  │      ╲─────────> 
  └─────────────────> 时间
```

**特征**：
1. 一直向下
2. 向下炒
3. 散户被迫跟供 = 冤枉钱
4. 股价向下，但**大股东不停赢钱**

#### 3️⃣ 喷井上升股（最佳！但要会沽）

```
股价
  ^      ╱╲
  │  ╱╲ ╱  ╲   ╱╲
  │ ╱  ╳    ╲ ╱  ╲   ← 一座座塔
  │╱        ╳    ╲
  └─────────────────> 时间
```

**特征**：
1. 一座座塔（走势）
2. 财技股**长揸必死**
3. 财技股**一定要沽的**
4. 每个塔顶都是离场点

### 0.8 配售分析实战清单

**收到供股公告时的分析步骤**：

- [ ] **Step 1**：计算大股东供股后持股比例
  - >75% = 供乾信号
  - <30% = 供钱信号
  
- [ ] **Step 2**：检查折让幅度
  - >20% + 大比例供股 = 第一口价卖出
  - <10% = 正常筹资

- [ ] **Step 3**：追踪5个重要日子
  - 标记除权日（最关键）
  - 设置R Report提醒

- [ ] **Step 4**：观察除权后走势
  - 贴近供股价横行 = 供乾可能
  - 快速反弹远离供股价 = 供钱可能

- [ ] **Step 5**：等待R Report
  - <30份申请 = **确认供乾**，准备买入
  - >100份申请 = 散户入场，观望

- [ ] **Step 6**：结合CCASS分析（读取 ccass-tracking 技能）
  - 检查大股东券商持仓变化
  - 验证是否真正在"收货"

### 1.2 拆股（Stock Split）

**定义**：增加股份数量，按比例降低每股价格。

**常见比例：** 1拆2、1拆5、1拆10

**目的分析：**
- ✅ 正当：提高流动性，降低入场门槛
- ⚠️ 可疑：合股后拆股
- ❌ 操纵：配合以"新"价格配售

### 1.3 合股（Reverse Split / Consolidation）

**定义**：减少股份数量，按比例提高每股价格。

**风险评估：**

| 比例 | 风险等级 | 常见目的 |
|------|----------|----------|
| 2合1 | 低 | 行政性 |
| 5合1 | 中 | 避免仙股 |
| 10合1 | 高 | 股价操纵准备 |
| 20合1+ | 非常高 | 即将大幅稀释 |

**警告模式：合股 → 配股 → 拆股**
```
步骤 1：10合1（价格：$0.10 → $1.00）
步骤 2：配股价 $0.50（"相对 $1.00"折让50%）
步骤 3：1拆5（价格：$0.50 → $0.10）
结果：大幅稀释，价格回到起点，资金已筹集
```

### 1.4 削減股本（Capital Reduction）

**定义**：减少股本，通常是为了消除累计亏损。

**类型：**
1. **股份註銷**：注销股份（减少总股数）
2. **削減面值**：降低面值（会计调整）
3. **回購註銷**：回购并注销

### 1.5 送紅股（Bonus Issue）

**定义**：从储备中向现有股东免费派发股份。

**分析框架：**

| 信号 | 解读 |
|------|------|
| 公司盈利豐厚 + 送紅股 | ✅ 正面：回饋股東 |
| 股價高企 + 送紅股 | ✅ 正面：降低入場門檻 |
| 無特別理由 + 送紅股 | ⚠️ 中性：可能為後續財技鋪路 |
| 配合配股/供股 | ❌ 負面：可能為稀釋做準備 |

### 1.6 分拆上市（Spin-off / Demerger）

**课程观点："天生的好股票"**

分拆上市往往是财技中的正面操作：
1. 释放隐藏价值（子公司估值重估）
2. 提高管理透明度
3. 为股东创造选择权
4. 母公司套现部分价值

**投资策略:**

| 階段 | 操作 | 理由 |
|------|------|------|
| 公告分拆 | 關注母公司 | 消息刺激 |
| 分拆詳情公布 | 評估子公司價值 | 定價關鍵 |
| 子公司上市前 | 考慮母公司 | 實物分派預期 |
| 子公司上市後 | 觀察估值差 | 套利機會 |

## 第二部分：买卖壳操作

### 2.1 全购（General Offer）

**定义**：以固定价格向全体股东发出收购要约。

**触发条件（《收購守則》规则26）：**
- 收购方跨越30%门槛
- 收购方在30-50%范围内，12个月内增持 >2%

**关键分析要点：**
- 收購價 vs 現價：溢价还是折让？
- 強制性全面收購：是否触发强制全面要约？
- 收購條件：先决条件
- 最低接納水平：最低接受比例

### 2.2 非全购（Partial Offer）

**定义**：仅对部分股份发出收购要约。

**关键特征：**
- 通常针对特定百分比（例如20%的股份）
- 需要執行人員裁決
- 通常暗示未来可能进行全面收购

### 2.3 白武士重組（White Knight Rescue）

**定义**：新投资者拯救财务困难的公司。

**机会识别（百倍股搖籃模式）：**

```
困境公司状态：
├── 股價暴跌（较高点 >80%）
├── 業務困難 / 債務危機
├── 停牌 or 除牌警告
└── 大股東尋求白武士

白武士入场：
├── 注入新資產（asset injection）
├── 削減股本清理帳目
├── 股本重組
└── 新管理層入主

潜在结果：
├── 成功重組 → 股價反彈數倍
└── 失敗 → 除牌
```

**尽职调查清单：**
- [ ] 白武士背景調查
- [ ] 注入資產質量
- [ ] 重組方案可行性
- [ ] 監管機構態度
- [ ] 時間表是否合理

### 2.4 啤殼上市（Backdoor Listing）

**定义**：私人公司收购上市公司以获得上市地位。

**关键指标：**
- 反向收購（Reverse Takeover）公告
- 重大资产注入
- 业务完全改变
- 新控股股东

**监管审查：**
- 港交所规则 14.06B：非常重大收购
- 规则 14.54：反向收购条款
- 可能停牌

## 第四部分：交易策略分析

### 4.1 向下炒（Downward Manipulation）

**定义**：大股东通过高位卖出、然后通过资本重组低价买回获利。

**模式识别：**
```
阶段 1：派发
├── 大股東高位配售
├── 利好消息出貨
└── CCASS持倉分散

阶段 2：股价下跌
├── 股價持續下跌
├── 成交量萎縮
└── 負面消息/業績差

阶段 3：累积
├── 大折讓供股/配股
├── 大股東低位增持
├── 削減股本重組
└── 資產注入

阶段 4：恢复
├── 股價反彈
├── 大股東獲利
└── 循環重複
```

### 4.2 出貨的藝術（Distribution Art）

**莊家散貨模型：**

| 阶段 | 行动 | CCASS 信号 |
|------|------|------------|
| 1 | 製造利好 | 持仓稳定 |
| 2 | 推高股價 | 轻微集中 |
| 3 | 分批出貨 | 逐渐分散 |
| 4 | 最後拉高 | 快速分散 |
| 5 | 急跌 | 散户券商仓位重 |

### 4.3 如何判轉勢？

**趨勢轉變的關鍵信號:**

| 信號類型 | 上升轉下跌 | 下跌轉上升 |
|---------|-----------|-----------|
| 價格形態 | 頂部形態（頭肩頂、雙頂） | 底部形態（頭肩底、雙底） |
| 成交量 | 高位放量滯漲 | 低位縮量企穩 |
| CCASS | 大戶減持、散戶承接 | 大戶累積、散戶拋售 |
| 公告 | 利好不漲、減持公告 | 利空不跌、增持公告 |

## 第五部分：进阶策略与法则

### 5.1 急插搏反弹 Y/N？（1日赚1倍潜力）

**核心问题**：股价急插后，应该搏反弹吗？

#### 急插搏反弹决策树

```
股价急插（单日跌>10%）
├── 检查1：急插原因
│   ├── 系统性风险（大市崩盘）→ ⚠️ 不博，等企稳
│   ├── 公司基本面恶化 → ❌ 不博，可能继续跌
│   ├── 短期消息利空 → ✅ 可以考虑
│   └── 无明显原因 → ✅ 财技股机会
│
├── 检查2：货源归边程度
│   ├── >90%归边 → ✅ 高概率反弹
│   ├── 70-90%归边 → ⚠️ 需要更多信号
│   └── <70%归边 → ❌ 散户多，反弹弱
│
├── 检查3：成交量变化
│   ├── 急插+放量 → ⚠️ 可能有大户出货
│   ├── 急插+缩量 → ✅ 洗盘可能性大
│   └── 急插+成交正常 → ✅ 可以考虑
│
└── 检查4：股价位置
    ├── 近历史低位 → ✅ 下跌空间有限
    ├── 中部位置 → ⚠️ 需要其他信号
    └── 高位急插 → ❌ 可能开始下跌趋势
```

#### 急插搏反弹的条件组合

**✅ 适合搏反弹的组合**：
```
条件1：货源归边 > 90%
条件2：成交缩量
条件3：股价近历史低位
条件4：无基本面恶化
条件5：大股东无减持

满足4/5条件 → 可以小仓位博反弹
```

**❌ 不适合搏反弹的组合**：
```
警告信号：
- 公司刚发盈警
- 大股东近期减持
- 货源不归边（<70%）
- 成交量放大
- 债务问题公告

出现任意2个 → 避免博反弹
```

#### 实战操作要点

| 阶段 | 操作 | 仓位控制 |
|------|------|---------|
| **试探** | 急插当日尾盘少量买入 | 5-10% |
| **确认** | 次日企稳加仓 | 再加10-15% |
| **获利** | 反弹10-20%开始减仓 | 逐步减持 |
| **止损** | 跌穿急插当日低位 | 立即止损 |

**⚠️ 风险提示**：急插搏反弹是高风险策略，不适合所有投资者。建议：
- 最多用总仓位的20%
- 设定严格止损（-5%至-10%）
- 1日内无反弹迹象则离场

### 5.3 混合财技详细分析

**定义**：多种财技操作的组合使用，目的是最大化大股东利益

#### 混合财技的常见组合

**组合1：合股 → 配股 → 拆股 (经典三部曲)**

```
阶段1：合股（例如10合1）
├── 目的：提高股价
├── 股价变化：$0.10 → $1.00
└── 效果：为高价配股做准备

阶段2：配股（大折让）
├── 目的：筹资+摊薄散户
├── 配股价：$0.50（相对$1.00折让50%）
├── 效果：散户被迫跟供或被摊薄
└── 大股东：低价收集散户弃权股

阶段3：拆股（例如1拆5）
├── 目的：恢复"正常"股价
├── 股价变化：$0.50 → $0.10
└── 效果：股价回到原点，但股本大增

📊 最终结果：
- 散户：被稀释，持股比例下降
- 大股东：完成融资，控制权加强
- 股价：回到原点，但基数已变
```

**组合2：向下炒 + 供股 (吸筹组合)**

```
阶段1：高位减持
├── 大股东在高位配售
├── 利好消息出货
└── CCASS显示持仓分散

阶段2：打压股价
├── 减少正面公告
├── 发布中性/负面消息
└── 成交量萎缩

阶段3：大折让供股
├── 供股价远低于市价
├── 条款不利散户
└── 无包销或不利包销

阶段4：低位收集
├── 大股东全额认购
├── 收集散户弃权股
├── 增加持股比例
└── 完成一轮"向下炒"
```

**组合3：白武士 + 削股 + 资产注入 (重组组合)**

```
困境公司
├── 股价暴跌（>80%）
├── 债务危机
└── 大股东套现离场
        ↓
白武士入场
├── 削减股本（清理亏损）
├── 合股（整理股价）
├── 资产注入（提供新业务）
└── 新管理层入主
        ↓
重组完成
├── 新业务启动
├── 股价反弹
└── 白武士获利

⚠️ 散户注意：
- 削股前持有 → 可能归零
- 削股后买入 → 有机会跟随上涨
```

#### 识别混合财技的信号

| 信号 | 含义 | 应对 |
|------|------|------|
| 短期内多次股本操作 | 可能是混合财技 | 分析整体意图 |
| 合股后很快配股 | 经典三部曲开始 | 警惕后续拆股 |
| 大股东频繁变更 | 可能买卖壳 | 研究新股东背景 |
| 业务完全改变 | 反向收购 | 评估新业务价值 |
| 削股+资产注入 | 重组 | 判断是真重组还是做局 |

### 5.4 收息股分析（高息股陷阱）

> **课程核心观点**：高息股是散户最大的认知陷阱之一

#### 高息股的本质

```
❌ 散户理解：
"我买100蚊股票，每年收3蚊息，被动收入！财务自由！"

✅ 实际情况：
"派息3蚊后，你的股票从100蚊变成97蚊"
"扣除税费和手续费后，你实际收到更少"
"总资产并没有增加，只是左口袋换右口袋"
```

**高息股的数学真相**

```
假设：股票价格100蚊，派息率5%

不派息情况：
├── 公司用5蚊再投资
├── 假设回报率20%
├── 年末股票价值：100 × 1.20 = 120蚊

派息情况：
├── 派息5蚊（扣税后约4.25蚊）
├── 股价自动调整至95蚊
├── 剩余93蚊继续投资（95-手续费）
├── 假设回报率20%
├── 年末股票价值：93 × 1.20 = 111.6蚊
├── 加回派息：111.6 + 4.25 = 115.85蚊

📉 派息导致你少赚：120 - 115.85 = 4.15蚊
```

#### 为什么公司派高息？

| 原因 | 对投资者的影响 |
|------|---------------|
| **增长放缓** | 公司找不到好的再投资机会 |
| **吸引散户** | 用"高息"包装平庸股票 |
| **大股东需要现金** | 通过派息提取公司价值 |
| **维持股价** | 用派息吸引买盘支撑股价 |
| **真正优质** | ⭐ 极少数情况，公司确实太赚钱 |

#### 高息股的正确评估

**✅ 真正的好高息股（极少）**：
- 行业龙头，护城河深
- 稳定现金流，年年赚钱
- 派息来自真实利润，不是举债
- 10年以上稳定派息记录
- 例子：部分公用事业股、REITs

**❌ 高息股陷阱（大部分）**：
- 股价长期下跌，被动"高息"
- 用借贷支撑派息
- 无真实盈利能力
- 派息不稳定
- 大股东持续减持

#### 高息股决策流程

```
见到"高息股"推荐
        ↓
问自己：这5%息是从哪来的？
        ↓
├── 来自稳定盈利 → 继续分析
└── 来自股价下跌 → ❌ 避开
        ↓
检查：过去5年派息记录
        ↓
├── 稳定或增长 → 继续分析
└── 不稳定或下降 → ❌ 避开
        ↓
检查：大股东行为
        ↓
├── 持续增持 → ✅ 可以考虑
└── 持续减持 → ❌ 避开
```

#### 课程金句

> "高息系你本身嘅钱，佢俾翻你咋"
> 
> "因为佢好，所以先派到高息；唔系因为高息，所以佢好"
> 
> "派高息 = 公司增长力下降的信号"

### 5.5 123法则（趋势反转判断）

**用途**：判断股价趋势是否反转的简单法则

#### 123法则的三个条件

```
趋势反转需要同时满足：

条件1️⃣：突破趋势线
├── 上升趋势：跌穿上升趋势线
└── 下降趋势：升穿下降趋势线

条件2️⃣：不创新高/新低
├── 上升转下跌：反弹不创新高
└── 下降转上升：回调不创新低

条件3️⃣：跌穿/升穿前低/前高
├── 上升转下跌：跌穿前一个调整低点
└── 下降转上升：升穿前一个反弹高点

✅ 三个条件全部满足 = 趋势反转确认
```

#### 123法则图解

**下降趋势转上升**：

```
股价
  ^
  │    2️⃣ 不创新低
  │         ↓
  │   ╲    ↗╱╲
  │    ╲  ╱    ╲    3️⃣ 升穿前高
  │     ╲╱      ╲  ↗
  │ 1️⃣突破下降线→  ╲╱  ✅ 确认反转
  │
  └─────────────────────> 时间
```

**上升趋势转下降**：

```
股价
  ^
  │           ╱╲
  │          ╱  ╲  2️⃣ 不创新高
  │         ╱    ╲   ↓
  │        ╱      ╲╱╲
  │       ╱   1️⃣突破↗ ╲
  │      ╱              ╲ 3️⃣ 跌穿前低
  │     ╱                ✅ 确认反转
  └─────────────────────> 时间
```

#### 123法则实战应用

| 场景 | 操作 | 备注 |
|------|------|------|
| 条件1满足 | 观望/减仓 | 第一个警告信号 |
| 条件1+2满足 | 准备行动 | 趋势弱化确认 |
| 条件1+2+3满足 | 执行交易 | 趋势反转确认 |

#### 123法则 + CCASS验证

```
123法则信号出现
        ↓
检查CCASS变化
        ↓
├── 下降转上升 + 大户累积 → ✅ 强信号
├── 下降转上升 + CCASS无变化 → ⚠️ 弱信号
├── 上升转下降 + 大户派发 → ✅ 强卖出信号
└── 上升转下降 + CCASS无变化 → ⚠️ 可能假跌破
```

#### 123法则的局限性

| 局限 | 应对方法 |
|------|---------|
| 滞后性 | 结合CCASS提前判断 |
| 假突破 | 等待确认，不急于行动 |
| 震荡市失效 | 在明确趋势中使用 |
| 主观性 | 统一标准，画线方法一致 |

## 第六部分：投资心理学

### 6.1 鎖定效應（Anchoring Effect）

**在財技分析中的應用:**

| 錨點陷阱 | 正確思維 |
|---------|---------|
| "股價曾經 $10，現在 $1 很便宜" | 看現在的基本面，而非歷史價格 |
| "折讓 20% 很吸引" | 折讓是相對什麼？合理價值是多少？ |
| "供股價比市價低" | 供股後的理論價值才是真正參考 |
| "大股東曾經 $5 買入" | 大股東的成本與你無關 |

### 6.2 損失規避（Loss Aversion）

**在財技分析中的影響:**

| 損失規避行為 | 後果 | 正確做法 |
|-------------|------|---------|
| 不願止損離場 | 越蝕越多 | 設定止損點並執行 |
| 跟隨供股避免攤薄 | 越買越多，風險集中 | 理性評估是否值得追加 |
| 接受蝕本價配售 | 認為"拎少好過冇" | 計算真實價值再決定 |
| 持有垃圾股等反彈 | 機會成本 | 認清現實，換馬 |
